**`Content-Length`** 개체 헤더는 수신자에게 보내지는, 바이트 단위를 가지는 개체 본문의 크기를 나타냅니다.

    </tr>
    <tr>
      <th scope="row">{{Glossary("Forbidden header name")}}</th>
      <td>yes</td>
    </tr>
  </tbody>

## 문법

```
Content-Length: <length>
```

## 디렉티브

- \<length>
  - : octets에 대한 십진수 단위의 길이.

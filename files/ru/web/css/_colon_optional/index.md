## Описание

CSS [псевдокласс](/ru/docs/Web/CSS/Псевдо-классы) `:optional` находит любые {{ HTMLElement("input") }} элементы, у которых не установлен атрибут {{ htmlattrxref("required","input") }}. Он позволяет формам легко отмечать необязательные поля, и давать им соответствующие стили.

Чтобы задать особый внешний вид обязательным полям формы можно использовать псевдокласс {{ cssxref(":required") }}.

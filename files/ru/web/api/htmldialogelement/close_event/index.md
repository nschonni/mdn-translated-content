{{ApiRef("HTML DOM")}}

Обработчик события для событий закрытия, отправляется в окно. (Не доступен в Firefox 2 или Safari)

## Синтаксис

```
window.onclose = funcRef;
```

### Параметры

- `funcRef` является ссылкой на функцию.

## Пример

```js
window.onclose = resetThatServerThing;
```

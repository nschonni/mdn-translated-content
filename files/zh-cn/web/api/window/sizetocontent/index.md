{{APIRef}}{{Non-standard_header}}

The **`Window.sizeToContent()`** 方法根据窗口内容调整窗口大小。为了使其工作，在调用此函数时应加载 DOM 内容，例如，一旦抛出 [`DOMContentLoaded`](/zh-CN/docs/Web/API/Window/DOMContentLoaded_event) 事件。

可以强行的调整窗口的最小尺寸。

## 语法

```plain
window.sizeToContent()
```

## 例子

```js
window.sizeToContent();
```

## 规范说明

此功能不是任何规范的一部分

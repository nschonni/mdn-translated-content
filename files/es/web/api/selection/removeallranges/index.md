### Resumen

Quita todos los rangos de la selección, dejando las propiedades [anchorNode](es/DOM/Selection/anchorNode) y [focusNode](es/DOM/Selection/focusNode) con un valor `null` (nulo) y dejando nada seleccionado.

### Sintaxis

```
sel.removeAllRanges();
```

### Parámetros

Ninguno.

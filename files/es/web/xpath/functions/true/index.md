La función `true` devuelve un valor booleano de verdadero.

### Sintaxis

```
true()
```

### Devuelve

El booleano `verdadero`.

### Definido

[XPath 1.0 4.3](http://www.w3.org/TR/xpath#function-true)

### Soporte en Gecko

Supported.

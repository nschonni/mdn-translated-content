## Resumo

Retorna o número de frames (tanto elementos `frame` ou `iframe`) dentro da janela.

## Sintaxe

```
quantidadeFrames = window.length;
```

- _quantidadeFrames_ é o número de frames.

## Exemplo

```js
if (window.length) {
  // este é um documento com subframes
}
```

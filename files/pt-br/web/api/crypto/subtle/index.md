{{APIRef("Web Crypto API")}}

A prorpiedade **`Crypto.subtle`** de apenas leitura retorna como objeto {{domxref("SubtleCrypto")}} permitindo exercer operações de criptografia.

## Sintaxe

```
var crypto = crypto.subtle;
```

{{domxref("Window")}} インターフェイスの **`visualViewport`** は読み取り専用プロパティーです。 特定のウィンドウの視覚的ビューポートを表す {{domxref("VisualViewport")}} オブジェクトを返します。

## 構文

```
var visualViewport = Window.visualViewport
```

### 値

{{domxref("VisualViewport")}} はオブジェクトです。

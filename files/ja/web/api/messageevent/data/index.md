{{APIRef("HTML DOM")}}

**`data`** は {{domxref("MessageEvent")}} インターフェイスのプロパティで、このメッセージの送信元が送信するデータを表します。

## 値

このメッセージの送信元が送信するデータです。これはどの型でも構いません。

## 例

```js
myWorker.onmessage = (e) => {
  result.textContent = e.data;
  console.log('Message received from worker');
};
```

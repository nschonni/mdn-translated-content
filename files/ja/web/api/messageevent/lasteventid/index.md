{{APIRef("HTML DOM")}}

**`lastEventId`** は {{domxref("MessageEvent")}} インターフェイスの読み取り専用プロパティで、このイベントの固有の ID を表す文字列です。

## 値

ID を表す文字列です。

## 例

```js
myWorker.onmessage = (e) => {
  result.textContent = e.data;
  console.log('Message received from worker');
  console.log(e.lastEventId);
};
```
